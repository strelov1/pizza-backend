## Pizza Frontend Part

In the project directory, you can run:

### `make up`

Runs local docker env <br />
Open [100.0.19.3](http://100.0.19.3) to view it in the browser.

### `make init`

Run migrations and init data <br />

### `make test`

Run tests <br />
